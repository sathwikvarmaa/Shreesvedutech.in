# ShreeSvEduTech.in

A responsive, multi-page educational consultancy website for guiding Indian students on MBBS abroad admissions.

## Overview

ShreeSvEduTech.in presents country-wise MBBS study options, eligibility criteria, service highlights, founder information, and a direct contact flow for student inquiries.

## Features

- Responsive landing page with mobile navigation
- Hero image carousel (Bootstrap)
- Country cards for MBBS destinations
- Eligibility criteria section
- About and founders sections
- Services and benefits sections
- Contact form with EmailJS integration
- WhatsApp quick-contact button
- Separate photo gallery page with image modal preview

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5
- Font Awesome
- Google Fonts (Poppins)
- EmailJS

## Project Structure

- `index.html` — Main landing page
- `Gallery.html` — Photo gallery page
- `style.css` — Main styles
- `Gallery.css` — Gallery page styles
- `script.js` — Contact form and EmailJS logic

## Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/sathwikvarmaa/Shreesvedutech.in.git
   ```
2. Open the project folder.
3. Launch `index.html` in your browser.
